# Caesar Cipher – Python Project 🔐

This is a beginner Python project that performs encryption and decryption using the Caesar Cipher algorithm.

## Features
- Encrypt messages
- Decrypt messages
- Handles spaces and non-alphabet characters
- Works with any shift value
- User interactive program

## Concepts Used
- Python functions
- While loops
- Conditionals
- String manipulation
- Modulo operator

## Example
hello → khoor (shift 3)
